项目名称：STM32 实验练习
设备：STM32F10x（本实验采用正点原子战舰开发板）、0.96IIC-OLED显示屏
功能：main函数中所写为始终，可在0.96OLED上显示星期、年月日、时分秒以及滚动显示“恭喜发财”。
可扩展功能：HARDWARE中设有蜂鸣器、GPIO口中断、窗口以及独立看门狗、LED灯、PWM驱动LED灯照明以及按键。
需要说明的是：本项目意在记录个人学习实验的程序，同时也开源所创建的文字图形动态显示方法，方便其他项目可直接调用。
动态 显示效果说明：1.滚动显示：可以调整左右滚动方向，也可调整速度，滚动内容需要自行到oled.h中调整。刷新速度为1Hz，要提高刷新率仅需在计时器中进行中断操作即可。
